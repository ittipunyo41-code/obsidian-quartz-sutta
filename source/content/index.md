ในสูตรว่า “อิยตมกิเอสานมนฺตสฺสโร ทีฆำ กฺวจิ ทิสสฺส คุณำ โทรำ สกฺขี จ”**การทำตัวรูป/วิธีที่ให้สำเร็จตัวรูป**  
การทำตัวรูป เพื่อให้สำเร็จรูปตามอุทาหรณ์ที่ตั้งไว้ วิธีการที่ใช้ก็มีหลากหลายวิธีด้วยกัน บางรูปใช้วิธี  
เดียว บางรูปก็ทำได้หลายวิธี และในการทำตัวรูปแต่ละตัวนั้น ต้องอาศัยวิธีการที่แตกต่างกันไป พอสรุปได้ดังนี้  

==๑. ใช้สูตรโดยตรง== เช่น เพราะสระหลัง แปลงนิคหิตเป็น มฺ และ ทฺ ใช้สูตรว่า “มทา สเร” เช่น  
“ตมหํ” มาจาก “ตํ + อหํ”, “เอตทโวจ” มาจาก “เอตํ +อโวจ” เป็นต้น  

==๒. ใช้วิธีการแบ่งสูตร== เรียกว่า “โยควิภาคะ” เช่น สูตรว่า “มทา สเร” ด้วยสูตรโดยตรง นิมิตเป็นเพราะ  
สระหลัง เมื่อแบ่งสูตรว่า “มทา” ตัดคำว่า “สเร” ทิ้งไป นิมิตก็ไม่จำกัดอยู่เฉพาะสระแล้ว ฉะนั้น เพราะพยัญชนะหลังก็สามารถแปลงนิคหิต เป็น มฺ ได้ เช่น “พุทฺธมฺ สรณมฺ คจฺฉามิ” มาจาก “พุทฺธำ สรณำ คจฺฉามิ”  

==๓. ใช้ จศัพท์==, วาศัพท์, ตุศัพท์, หรือ อนฺตศัพท์ เป็นต้น เช่น จศัพท์ในสูตรว่า “โท ธสฺส จ” สามารถแปลง ธ เป็น ห ได้ เช่น “สาหุ ทสฺสนํ” มาจาก “สาธุ ทสฺสนํ”, วาศัพท์ในสูตรว่า “วคฺคนฺตำ วา วคฺเค” สามารถแปลงนิคหิต  
เป็น ลฺ ได้ เช่น “ปุลฺลิงฺคํ” มาจาก “ปุม + ลิงฺคํ”, ตุศัพท์ในสูตรว่า “พฺรหฺมโต ตุ สฺมึ นิ” สามารถแปลง สฺมึวิภัตติ เป็น นิ ได้ แม้หลังจากกมฺมศัพท์ จมฺมศัพท์ และมุทฺธศัพท์ เป็นต้น เช่น “กมฺมนิ” มาจาก “กมฺม +สฺมึ”,  
“จมฺมนิ” มาจาก “จมฺม + สฺมึ” และ “มุทฺธนิ” มาจาก “มุทฺธ + สฺมึ” เป็นต้น, อนฺตศัพท์ในสูตรว่า “ปุมนฺตสฺสา สิมฺหิ” เพราะ สิวิภัตติ สามารถแปลงที่สุดของ มฆวศัพท์ และยุวศัพท์ เป็นต้น พร้อมกับสิวิภัตติเป็น อา ได้ เช่น “มฆวา” มาจาก “มฆว + สิ” เป็นต้น

==๔. ใช้วิธีที่แสดงนัยในสูตร== เช่น ภาวนิเทศ, แสดงรูปพหุวจนะ เช่น “กาสตฺต” ภาวนิเทศในสูตรว่า  
“กรสฺส กาสตฺตมชฺชตนิมฺหิ” ลง สฺอาคม เช่น “อคมาสิ” มาจาก “อ+คมุ+สฺ+อี”, ด้วยการแสดงเป็นพหุวจนะว่า “มหตํ”ในสูตรว่า “มหตํ มหา ตุลฺยาธิกรเณ ปเท” สามารถแปลง มหนฺตศัพท์ เป็น มห ได้ เช่น  
“มหพฺภยํ”มาจาก “มหนฺต+ภย” เป็นต้น  

==๕. ใช้นิปาตนะ วิธีการทำตัวรูปโดยใช้รูปสำเร็จที่มีปรากฏอยู่ในสูตร== เช่น แปลง อิม เป็น อิ ด้วย อินิปาตนะ  
 เช่น “อีทิโส” ศัพท์เดิมมาจาก  
“อิมสทฺทูปปท+ทิสธาตุ+กฺวิปัจจัย” เป็นต้น  

==๖. ในกรณีที่ไม่มีสูตรทำจริง ๆ ก็จะใช้มหาสูตร== ซึ่งเป็นสูตรที่มีหน้าที่มาก มีอยู่ ๔ สูตร คือ  
๖.๑ เตสุ วุทฺธิโลปาคมวิการวิปรีตาเทสา จ. (รูปสิทธิ. ตัทธิต สูตรที่ ๓๗๐)  
๖.๒. ยทนุปปนฺนา นิปาตนา สิชฺฌนฺติ. (รูปสิทธิ. ตัทธิต สูตรที่ ๔๒๓)  
๖.๓. กฺวจิ ธาตุวิภตฺติปฺปจฺจยานำ ทีฆวิปรีตาเทสโลปาคมา จ. (รูปสิทธิ. อาขยาต สูตรที่ ๔๘๘)  
๖.๔. ปจฺจยา ทนิฏฺฐา นิปาตนา สิชฺฌนฺติ. (รูปสิทธิ. กิพพิธาน สูตรที่ ๖๒๔)  

==๗. ใช้นิรุตตินัย มี ๕ ประการ== คือ==๗.๑ วณฺณาคม การลงอักษรใหม่== เช่น “นกฺขตฺตราชาริว ตารกานํ”  
ศัพท์เดิมมาจาก “นกฺขตฺตราชาอิว ตารกานํ”ลง รฺอาคม ด้วยสูตร “ยวมทนตรลา จาคมา”==๗.๒ วณฺณวิปริยย การสลับที่อักษร (การกลับหน้ากลับหลังอักษร)== เช่น “สตฺเต หึสตีติ สีโห”  
(หึส หึสายำ ในการเบียดเบียน อ) เมื่อควรจะเป็น “หึโส” เพราะมาจาก หึสธาตุ แต่ก็กลับหน้ากลับหลังเป็น  
สึห แล้วลบนิคหิต และทำทีฆะ==๗.๓ วณฺณวิการ การเปลี่ยนแปลงอักษร== เช่น “นิยโก” ศัพท์เดิมเป็น “นิชโก” แปลง ช เป็น ย==๗.๔ วณฺณนาส การลบอักษร เช่น เมขลา== มาจากรูปวิเคราะห์ว่า “เมหนสฺส ขสฺส มาลา  
เมขลา” ศัพท์เดิมมาจาก “เมหน+ข+มาลา” โดยลบ “หน” และ “มา” สำเร็จรูปเป็น “เมขลา”==๗.๕ ธาตุอตฺถาติสยโยค การเพิ่มอรรถพิเศษของธาตุ== เช่น“มหิยำ รวตีติ มยูโร” (ย่อมรื่นรมย์ บนแผ่นดิน เพราะเหตุนั้น ชื่อว่ามยูร. (นกยูง) คำว่า “มยูโร” มาจาก “มหีสทฺทูปปท+รุ สทฺเท ในการทำเสียง+อปัจจัย” แต่ รุธาตุ ยังสามารถกล่าวอรรถ รมณ (รื่นรมย์, ยินดี) ได้อีก ซึ่งเป็นอรรถที่เพิ่มเข้ามา การที่ รุธาตุกล่าวอรรถพิเศษเพิ่มเติมได้นั้น เรียกว่า “อตฺถาติสย”  

  

==ย-ว-ม-ทาทิสุตฺเตน อฏฺฐ พฺยญฺชนมาคมา  
จสทฺเทน ปน เสสา จตุวีสติ พฺยญฺชนา.== (กจฺจายนวณฺณนา หน้า ๕๓)  
พยัญชนะอาคม ๘ ตัว คือ ย, ว, ม, ท, น, ต, ร และ ล ย่อมถูกถือเอา  
ด้วยสูตรว่า “==[[กัจจายสูตรแปล]]==” ส่วนพยัญชนะ ๒๔ ตัวที่เหลือ  
ย่อมถูกถือเอาด้วย จศัพท์

คห ชุต ญา สิ สุ สมฺภู สร และ สสธาตุ เป็นต้น

ตึส และวตศัพท์ เป็นต้น  

==(๓) วตุ-วฏุ-ทิสาทีนมนฺตญฺจ.== หลังรัสสสระและสระอา ให้ทำเทฺวภาวะพยัญชนะตัวหลังของ วตุ วฏุ และทิส  
ธาตุ เป็นต้น

อุปสัค ต จตุ ฉ และ ส ที่แปลงมาจาก สนฺตศัพท์ เป็นต้น  

==(๕) อปทนฺตานาการทีฆโต ยการาทิ จ.== ให้ทำเทฺวภาวะ ยอักษร เป็นต้น ที่อยู่หลังทีฆสระซึ่งไม่ใช่ที่สุด  
ของบท และไม่ใช่สระอา  

==(๖) ยวตํ ตลนาทีน มาเทโส จ สยาทินํ  
สหธาตฺวนฺตยาเทโส สีสกาโร ตปาทิโต.==

ข. เมื่อมีการแปลงพยัญชนะที่ประกอบกับ ย ด้วยสูตร “เอ เห ญํ” และ “ส เย จ” แล้วให้ทำ  
เทฺวภาวะ  
ค. เมี่อมีการแปลง ย กับที่สุดธาตุ ด้วยสูตร “ตสฺส จวคฺคฯ” และ “ตถา กตฺตริ จ” แล้วให้ทำ  
เทฺวภาวะ  
ง. ให้ทำเทฺวภาวะ สฺ ของ สีปัจจัยที่อยู่หลัง ตปศัพท์ เป็นต้น  

==(๗) ฉนฺทานุรกฺขเณ จ.== ในที่ต้องการรักษาฉันท์ ให้ทำเทฺวภาวะได้  

==(๘) ฆร-เฌ-ธํสุ-ภมาทีนมาทิ จ.== หลังจากรัสสสระและสระอา ให้ทำเทฺวภาวะพยัญชนะตัวหน้าของ ฆร  
เฌ ธํสุ และ ภมุธาตุ เป็นต้น  

==(๙) รสฺสาการโต วคฺคานํ จตุตฺถทุติยา จ.== หลังจากรัสสสระและสระอา ให้ทำเทฺวภาวะพยัญชนะตัวที่ ๑  
และตัวที่ ๓ ในวรรค ด้วยสูตร “วคฺเค โฆสาโฆสานํ ตติยปฐมา”

#  

ทำตัวรูป ==โลกคฺคปุคฺคโล== ศัพท์เดิมคือ = โลก + อคฺคปุคฺคโล  
แยกพยัญชนะ กฺ ออกจากสระ อ โลกฺ อ + อคฺคปุคฺคโล = ==[[กัจจายสูตรแปล]]==  
เพราะสระหลัง ลบสระหน้า โลกฺ อ + อคฺคปุคฺคโล = ==[[กัจจายสูตรแปล]]====.==  
นำพยัญชนะ กฺ ไปประกอบกับสระ อ โลกคฺคปุคฺคโล = ==[[กัจจายสูตรแปล]]====.==  
สำเร็จรูปเป็น โลกคฺคปุคฺคโล. (บุคคลผู้เลิศกว่าชาวโลก)  

ทำตัวรูป ==ปญฺญินฺริยํ== ศัพท์เดิมคือ = ปญฺญา + อินฺทฺริยํ  
แยกพยัญชนะ ญฺ ออกจากสระ อา ปญฺญฺ อา + อินฺทฺริยํ = ==[[กัจจายสูตรแปล]]==  
เพราะสระหลัง ลบสระหน้า ปญฺญฺ อา + อินฺทฺริยํ = ==[[กัจจายสูตรแปล]]====.==  
นำพยัญชนะ ญฺ ไปประกอบสระ อิ ปญฺญินฺทฺริยํ = ==[[กัจจายสูตรแปล]]====.==  
สำเร็จรูปเป็น ปญฺญินฺทฺริยํ. (อินทรีย์คือปัญญา)  

ทำตัวรูป ==ตีณิมานิ== ศัพท์เดิมคือ = ตีณิ + อิมานิ  
แยกพยัญชนะ ณฺ ออกจากสระ อิ ตีณฺ อิ + อิมานิ = ==[[กัจจายสูตรแปล]]==  
เพราะสระหลัง ลบสระหน้า ตีณฺ อิ + อิมานิ = ==[[กัจจายสูตรแปล]]====.==  
นำพยัญชนะ ณฺ ไปประกอบกับสระ อิ ตีณิมานิ = ==[[กัจจายสูตรแปล]]====.==  
สำเร็จรูปเป็น ตีณิมานิ. (สาม เหล่านี้)

  

ทำ ตัวรูป ==มาตุปฏฺฐานํ== ศัพท์เดิมคือ = มาตุ + อุปฏฺฐานํ  
แยกพยัญชนะ ตฺ ออกจากสระ อุ มาตฺ อุ + อุปฏฺฐานํ = ==[[กัจจายสูตรแปล]]==  
เพราะสระหลัง ลบสระหน้า มาตฺ อุ + อุปฏฺฐานํ = ==[[กัจจายสูตรแปล]]====.==  
นำ พยัญชนะ ตฺ ไปประกอบกับสระ อุ มาตุปฏฺฐานํ = ==[[กัจจายสูตรแปล]]====.==  
สำ เร็จรูปเป็น มาตุปฏฺฐานํ. (การบำ รุงมารดา)  

ทำ ตัวรูป ==สพฺเพว== ศัพท์เดิมคือ = สพฺเพ + เอว  
แยกพยัญชนะ พฺ ออกจากสระ เอ สพฺพฺ เอ + เอว = ==[[กัจจายสูตรแปล]]==  
เพราะสระหลัง ลบสระหน้า สพฺพฺ เอ + เอว = ==[[กัจจายสูตรแปล]]====.==  
นำ พยัญชนะ พฺ ไปประกอบกับสระ เอ สพฺเพว = ==[[กัจจายสูตรแปล]]====.==  
สำเร็จรูปเป็น สพฺเพว. (ทั้งปวงนั่นเทียว)

  

ทำตัวรูป ==ยสฺสทานิ== ศัพท์เดิมคือ = ยสฺส + อิทานิ  
หลังจากสระหน้า ลบสระ อิ หลัง ยสฺส + อิ ทานิ = ==[[กัจจายสูตรแปล]]==  
สำเร็จรูปเป็น ยสฺสทานิ. (แห่งการไปใด ในบัดนี้)

  

ทำตัวรูป ==ฉายาว== ศัพท์เดิมคือ = ฉายา + อิว  
หลังจากสระหน้า ลบสระ อิ หลัง ฉายา + อิ ว = ==[[กัจจายสูตรแปล]]==  
สำเร็จรูปเป็น ฉายาว. (เพียงดังเงา, เหมือนดังเงา)

นำพยัญชนะ วฺ ไปประกอบกับสระ เอ เสฺวว = ==[[กัจจายสูตรแปล]]====.==

ทำตัวรูป ==อพฺโภกาโส== ศัพท์เดิมคือ = อภิ + โอกาโส  
เพราะสระหลัง แปลง อภิ เป็น อพฺภ อภิ อพฺภ + โอกาโส = ==[[กัจจายสูตรแปล]]==  
แยกพยัญชนะ ภฺ ออกจากสระ อ อพฺภฺ อ + โอกาโส = ปุพฺพมโธฐิตมสฺสรํ สเรน วิโยชเย.  
เพราะสระหลัง ลบสระหน้า อพฺภฺ อ + โอกาโส = สรา สเร โลปํ.  
นำพยัญชนะ ภฺ ไปประกอบกับสระ โอ อพฺโภกาโส = นเย ปรํ ยุตฺเต.  
สำเร็จรูปเป็น อพฺโภกาโส. (ที่ว่าง, กลางแจ้ง)  

ตัวอย่างที่มีวิธีทำตัวรูปเหมือนกับ “อพฺภกฺขานํ” มีดังต่อไปนี้  
อพฺภีริตํ = อภิ + อีริตํ (กล่าวยิ่งแล้ว, พูดพล่าม)  
อพฺภุทีริตํ = อภิ + อุทีริตํ (กล่าวยิ่งแล้ว, พูดพล่าม)

ทำตัวรูป ==อชฺโฌคาเหตฺวา== ศัพท์เดิมคือ = อธิ + โอคาเหตฺวา

ทำตัวรูป ==เอกมิทาหํ== ศัพท์เดิมคือ = เอกมิธ + อหํ  
เพราะสระหลัง แปลง ธ เป็น ท เอกมิ ธ → ท + อหํ = ==[[กัจจายสูตรแปล]]==  
แยกพยัญชนะ ทฺ ออกจากสระ อ เอกมิ ทฺ อ + อหํ = ปุพฺพมโธฐิตมสฺสรํ สเรน วิโยชเย.  
เพราะสระหลัง ลบสระหน้า เอกมิ ทฺ ~~อ~~ + อหํ = สรา สเร โลปํ.  
เมื่อลบสระหน้าแล้ว ทีฆสระหลัง เอกมิ ทฺ + อ→อา หํ = ทีฆํ.  
นำพยัญชนะ ทฺ ไปประกอบกับสระ อา เอกมิทาหํ = นเย ปรํ ยุตฺเต.  
สำเร็จรูปเป็น เอกมิทาหํ. (สมัยหนึ่ง เรา ... ในที่นี้)  

ทำตัวรูป ==สาหุ ทสฺสนํ== ศัพท์เดิมคือ = สาธุ + ทสฺสนํ  
แยกพยัญชนะ ธฺ ออกจากสระ อุ สา ธฺ อุ + ทสฺสนํ = ปุพฺพมโธฐิตมสฺสรํ สเรน วิโยชเย.  
เพราะสระหลัง แปลง ธฺ เป็น หฺ สา ธฺ→หฺ อุ + ทสฺสนํ ===[[กัจจายสูตรแปล]]==  
นำพยัญชนะ หฺ ไปประกอบกับสระ อุ สาหุ ทสฺสนํ = นเย ปรํ ยุตฺเต.  
สำเร็จรูปเป็น สาหุ ทสฺสนํ. (การเห็น ยังประโยชน์ให้สำเร็จ)

  

ทำตัวรูป ==ยถริว== ศัพท์เดิมคือ = ยถา + เอว  
แยกพยัญชนะ ถฺ ออกจากสระ อา ยถฺ อา + เอว = ปุพฺพมโธฐิตมสฺสรํ สเรน วิโยชเย.

นำพยัญชนะ ถฺ ไปประกอบกับสระ อ ยถริว = นเย ปรํ ยุตฺเต.  
สำเร็จรูปเป็น ยถริว. (ฉันใดนั่นเทียว)
